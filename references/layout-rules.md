# Layout Rules

## Safe Defaults

- Slide ratio: 16:9
- Outer margins: 0.45in minimum
- Gutter between major regions: 0.18in minimum
- Prefer 2-column layouts over 3-column text-heavy layouts
- Keep at least 15% vertical slack inside body text regions

## Text Density

- Mixed Chinese-English text wraps earlier than English-only text
- Compact cards should not contain more than 2 wrapped lines
- Titles and subtitles need extra width and height reserve
- If a chart needs a long note, place the note beside the chart rather than below it

## Chart Guidance

- Multi-panel charts work well for result slides
- Use direct data labels sparingly
- Avoid legends that force the chart to shrink too much
- Leave bottom room for x-axis labels and annotations

## Repair Order

1. Shorten copy
2. Widen box
3. Increase height
4. Reduce font size slightly
5. Redesign layout
